# AIWF v1.6.1 Release Note

## Positioning
v1.6.1 is the closure hardening and dataset readiness baseline.

## Added
- `check --finalize-ready`
- `report --format json|markdown`
- normalized evidence event schema `aiwf-event-v0.2`

## Changed
- finalized tasks reject new evidence records
- post-finalize evidence pollution is detected
- finalize appends implicit transition to finalized
- finalize rolls back metadata if index sync fails

## Deferred
- strict phase-gated finalize
- full strict phase state machine
- report ranking grouped by schema/version
- refined blocked metrics

## Compatibility
- legacy `event` + string `result` records remain readable
- no legacy mass rewrite/backfill is performed
