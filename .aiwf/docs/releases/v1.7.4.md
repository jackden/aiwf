  - `.aiwf/metadata.local.env`
  - `.aiwf/metadata.current`

- No runtime behavior changes.
- No workflow semantic changes.
- No protocol semantic changes.
- No finalize/check/guard behavior changes.
- No metadata feature additions.

## Version Metadata Policy
For the current lightweight AIWF project, release version, tool version, and workflow protocol version remain intentionally aligned at `1.7.4`.

Patch-level metadata alignment and packaging hygiene do not by themselves imply:
- runtime semantic changes
- event schema changes
- finalize gate changes
- diagnostic behavior changes

## Deferred / Non-goals
- No runtime upgrade beyond existing v1.7.4 implementation.
- No workflow protocol upgrade.
- No metadata subsystem feature expansion.
